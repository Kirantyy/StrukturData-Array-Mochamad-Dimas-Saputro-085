Buatkan program Linked List Mahasiswa dalam bahasa Python dan Java dengan menu interaktif di terminal.

Spesifikasi:
- Setiap node menyimpan:
  - NIM
  - Nama
  - Pointer ke node berikutnya
- Gunakan konsep Linked List (bukan array/list bawaan sebagai penyimpanan utama).
- Tampilkan menu seperti berikut:

=== MENU ===
1. Tambah data di awal
2. Tambah data di akhir
3. Hapus data berdasarkan NIM
4. Tampilkan semua data
5. Keluar

Tambahkan informasi jumlah data pada tampilan menu, contoh:
=== MENU PYTHON (Data: 0/10) ===

Tambahkan validasi maksimal data 10 mahasiswa.

Berikan comment pada bagian kode agar saya dapat memahami source code tersebut.
